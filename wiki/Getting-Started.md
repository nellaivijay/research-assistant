# Getting Started Guide

Welcome to the Enhanced Research Assistant! This guide will help you get started with using the application.

## 🚀 Quick Start

### 1. Access the Application

Visit the live application at: [https://huggingface.co/spaces/nellaivijay/research-assistant](https://huggingface.co/spaces/nellaivijay/research-assistant)

### 2. Basic Workflow

The Research Assistant helps you discover, analyze, and manage research papers efficiently.

#### Step 1: Find Papers
- Go to the **🔍 Paper Recommendations** tab
- Enter a paper ID (Semantic Scholar, arXiv, DOI) or URL
- Click **"Get Recommendations"** to discover related papers

#### Step 2: Analyze Papers
- Review the recommended papers
- Check citation counts, impact scores, and relevance
- Use **📊 Citation Analysis** tab for detailed metrics

#### Step 3: Build Reading List
- Select interesting papers
- Add them to your reading list in **📖 Reading List** tab
- Organize by priority or topic

#### Step 4: Take Notes
- Use **📝 Notes & Annotations** tab to add insights
- Attach notes to specific papers
- Track your reading progress

#### Step 5: Export Results
- Use **📤 Export** tab to export your reading list
- Choose from Markdown, JSON, or BibTeX formats
- Import into your reference manager

## 🤖 Using AI Features

### AI Model Selection

The Research Assistant supports 15+ AI models for paper analysis:

#### Free Models
- **Rule-Based Analysis**: Basic analysis without API costs
- **Hugging Face Models**: Mistral 7B, Llama 3 8B, Gemma 7B
- **Ollama Models**: Llama 3, Mistral, Mixtral (local)

#### Paid Models
- **OpenAI**: GPT-4o Mini, GPT-4o
- **Anthropic**: Claude 3 Haiku, Claude 3 Sonnet
- **Google**: Gemini Pro, Gemini 1.5 Pro

### Setting Up API Keys

1. Go to **🤖 Model Selection** tab
2. Select your preferred service (OpenAI, Anthropic, Google)
3. Enter your API key
4. Click **"Save API Key"**

### AI-Powered Analysis

Once API keys are configured:

1. **Model Comparison**: Compare two models side-by-side
2. **Batch Processing**: Analyze multiple papers at once
3. **Custom Prompts**: Create custom analysis prompts
4. **Auto Model Selection**: Let the system choose the best model
5. **A/B Testing**: Compare model performance on your papers

## 🧪 A/B Testing

A/B testing helps you find the best AI model for your research:

1. Go to **🧪 A/B Testing** tab
2. Enter paper details in JSON format
3. Select Model A and Model B
4. Choose analysis type
5. Click **"Run A/B Test"**
6. Compare results and record preferences
7. Build statistics over time

## 🔧 Troubleshooting

### Common Issues

**No recommendations found**
- Try a different paper ID or URL
- Check that the paper exists in the database
- Use a more well-known paper as seed

**AI analysis not working**
- Verify API keys are correctly saved
- Check your API credits/balance
- Try a different model or service
- Use rule-based analysis as fallback

**Export format issues**
- Ensure you have papers in your reading list
- Check the selected export format
- Try a different format if needed
